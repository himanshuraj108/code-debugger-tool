Features

  • Code Debugging: Debug code in multiple languages and view output.
  • Execution Visualization: Visualize code execution steps (Python only).
  • Question Generation: Automatically generate questions based on code.
  • Auto-correct: Automatically correct code errors and provide suggestions.
  • Clipboard Integration: Copy and paste code directly from clipboard.
  • Watch History: Save and manage a history of code executions.
  • Interactive Interface: Toggle features like visualizer and question generation.
